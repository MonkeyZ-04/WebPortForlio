สวัสดีครับผมนนท์ปวิธ สูงชัยภูมิ ผมหลงใหลในเทคโนโลยีมาโดยตลอดและวิธีที่สามารถนำมาใช้แก้ปัญหาและทำให้ชีวิตของผู้คนง่ายขึ้น ผมมีความถนัดด้านคณิตศาสตร์และวิทยาศาสตร์เป็นอย่างมาก และผมชอบเรียนรู้ว่าคอมพิวเตอร์ทำงานอย่างไรและจะตั้งโปรแกรมอย่างไร ปัจจุบันผมกำลังการศึกษาโรงเรียนเตรียมอุดมศึกษาพัฒนาการ ผมได้ใช้เวลาว่างของผมได้ไปเรียนเกี่ยวกับภาษาต่างๆ เช่น Python Kotlin และ C++ ที่เป็ภาษาโปรแกรม รวมถึงภาษามาร์กอัปเช่น HTML CSS นอกจากนี้ ผมยังผ่านการเข้าค่ายแข่งขันหลายครั้ง ซึ่งผมได้รับประสบการณ์จริงอันมีค่าในการทำงานในโครงการที่มีแอปพลิเคชัน ผมกระตือรือร้นที่จะทำงานในฐานะวิศวกรคอมพิวเตอร์หลังจากสำเร็จการศึกษา ผมสนใจที่จะทำงานให้กับบริษัทที่เกี่ยวกับนวัตกรรมทางเทคโนโลยี ซึ่งผมสามารถใช้ทักษะและความรู้ที่มีเพื่อช่วยในการออกแบบและพัฒนาระบบและโซลูชั่นที่ทันสมัย ผมเป็นคนที่ขยันขันแข็งและมองหาการเรียนรู้และพัฒนาอยู่เสมอ ผมเป็นคนที่สนุกกับการทำงานร่วมกับผู้อื่นเพื่อแก้ปัญหาและบรรลุเป้าหมายร่วมกัน ผมยังมีความสามารถในการวิเคราะห์และสนุกกับการใช้ทักษะการแก้ปัญหาเพื่อรับมือกับความท้าทายต่างๆครับ
