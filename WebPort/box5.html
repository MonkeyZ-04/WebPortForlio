เข้าร่วมค่าย IT Camp18 (Data Sci KMITL) เข้าร่วมค่าย IT Camp18 โดยอยู่หัวข้อ Data Sci ได้ทำการสร้าง Model เกี่ยวกับ ยา ซึ่งจะเป็น Model ที่ช่วยจ่ายยาให้ผู้ป่วย ซึ่งทางกลุ่มของผมได้ทำตัวModel ออกมาทั้งหมด 3 ตัว ซึ่งผมได้รับหน้าที่ทำ Model unsupervised learning ที่เป็น Clustering เอาไว้จำแนกประเภทของผู้ป่วย โดยที่Model แบบ Clustering จะแบ่งกลุ่ม
ของผู้ป่วยออกมาเป็น 6 แบบ ที่แตกต่างกัน ตัว Model จะดูจากค่า Cholesterol, Blood pressure, Age, Gender, Na to K และนำมาจำแนกผู้ป่วย ซึ่งทางกลุ่มของผมได้ทำ Model ที่เป็น Supervised อีก 2 ตัว คือ KNN Decision Tree โดย 2 Model มีค่าความแม่นยำอยูที่ 55% และ 98.33% ตามลำดับ ซึ่งก่อนที่จะมเป็นตัว Model ได้ ผมทำขัั้นตอน CRISP-DM เพื่อเป็นการเตรียมข้อมูลให้พร้อมใช้งาน อธิเช่น เปลี่ยนข้อความเป็นตัวเลขเพื่อให้เอาทำ Model ต่อได้ และคัดเอา ค่า null ออก
