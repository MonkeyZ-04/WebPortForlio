ได้รางวัล The Best Creative โดยการนำเสนอโครงงาน Application สำหรับถ่ายรูปยาสำหรับเด็ก(ยาน้ำ) Quack ในหัวข้อน้ำ
ที่มาของApp นี้คือการที่เด็กนั้นมีความอ่อนไหวในการใช้ยา เด็กน้ำหนักต่างกัน ต้องได้รับประทานยาที่ต่างกัน อีกทั้งยังเป็นการรักษาหลังการรักษาอีกทีด้วย เพราะเภสัชหรือบุคคลกรทางการแพทย์ ทำงานอย่างหนักอาจทำให้เกิดข้อผิดพลาดได้ ทางผมเลยคิดApp ตัวนี้ขึ้นมาแก้ปัญหาครับ โดยที่จะมี Image processing โดยใช้ Tensorflow ในการทำ Model ตรวจจับยา โดยที่ตัวของ Model ได้ใช้รูป 1500 รูปในการเทรน เพื่อที่จะได้ตัวModel ที่มีประสิทธิภาพมา ในตัวของแอพ จะมีกล้องเอาไว้ถ่ายรูปยา
แล้วระบบจะดึงข้อมูลจาก Firebase บอกรายละเอียดของยา ทั้งวิธีการกิน การกินต่อวัน รวมถึง ผลข้างเคียง ยังมีปฏิทินที่เอาไว้บอกว่ากินยาไปแล้วกี่วัน เพื่อที่จะให้ผู้ใช้ได้ประเมินตนเองว่า อาการดีขึ้นรึยัง อาการตอนนี้เป็นยังบ้างเทียบกับวันแรกที่เป็น มีช่องเอาไว้ค้นหายา ซึ่งช่องนี้จะนำข้อมูลจาก Firebase มาแสดงให้ผู้ใช้ดู และยังมีนาฬิกาปลุก เตือนว่าต้องกินยาแล้ว เพื่อช่วยให้ผู้ใช้ได้รับประทานยาได้อย่างตรงเวลา โดยผมได้รับหน้าที่เป็น developer หลักของกลุ่ม
